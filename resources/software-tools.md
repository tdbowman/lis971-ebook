# Software and Tools Reference

This page lists the software tools available for your dissertation research, including both institutional licenses and free alternatives.

---

## Citation Management

| Tool | Description | Access |
|------|-------------|--------|
| **[Zotero](https://www.zotero.org/)** | Free, open-source citation manager | zotero.org (free) |

```{tip}
Dominican University has a [Zotero guide](https://research.dom.edu/citationguides/zotero) to help you get started. Zotero integrates with Word, Google Docs, and LibreOffice for seamless APA 7th edition citation formatting.
```

## Statistical Analysis

| Tool | Description | Access |
|------|-------------|--------|
| **SPSS** | Industry-standard statistical software | DU Virtual Lab |
| **[PSPP](https://www.gnu.org/software/pspp/)** | Free, open-source alternative to SPSS | gnu.org (free) |
| **[R](https://www.r-project.org/)** / **[RStudio](https://posit.co/)** | Open-source statistical computing environment | Free |
| **[Python](https://www.python.org/)** (scipy, pandas, statsmodels) | Open-source data analysis and visualization | Free |

```{admonition} DU Virtual Lab Access
:class: note
SPSS is available through the DU Virtual Lab at [remoteapp.dom.edu](http://remoteapp.dom.edu/). You must be connected to the GlobalProtect VPN. See [VPN setup instructions](https://support.dom.edu/TDClient/2074/Portal/KB/ArticleDet?ID=133930).
```

## Qualitative Data Analysis (QDA)

| Tool | Description | Access |
|------|-------------|--------|
| **NVivo** | Industry-standard QDA software; powerful query and visualization tools | DU Virtual Lab |

| **[Dedoose](https://www.dedoose.com/)** | Web-based QDA; good for mixed methods and team coding | dedoose.com (subscription) |
