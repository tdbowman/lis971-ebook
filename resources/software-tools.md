## Free Statistics Learning Resources

If statistics feels intimidating, you're not alone-and these free resources can help. They're organized from "I need to start from zero" to "I need help with a specific advanced topic."

---

### Video Courses and Tutorials

| Resource | What It Covers | Best For |
|----------|---------------|----------|
| **[Khan Academy: Statistics & Probability](https://www.khanacademy.org/math/statistics-probability)** | 16 modules from basic data display through inference and significance testing, with interactive exercises and quizzes | Starting from scratch; building foundational understanding at your own pace |
| **[Crash Course Statistics (YouTube)](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNM_Y-bUAhblSAdWRnmR9eR)** | 44 short, animated videos covering descriptive stats, probability, distributions, regression, ANOVA, and chi-square | Visual learners who want fast, entertaining overviews of key concepts |
| **[Harvard Statistics 110 (YouTube)](https://www.youtube.com/playlist?list=PL2SOU6wwxB0uwwH80KTQ6ht66KWxbzTIo)** | 34 university lectures on probability and statistics from Prof. Joe Blitzstein | Deeper mathematical foundations; more rigorous than Khan Academy |
| **[StatQuest with Josh Starmer (YouTube)](https://www.youtube.com/c/joshstarmer)** | Individual videos on specific tests and concepts-t-tests, regression, PCA, machine learning basics-explained with simple drawings | When you need one specific concept explained clearly; great for "what IS a p-value?" moments |
| **[Coursera: Basic Statistics (University of Amsterdam)](https://www.coursera.org/learn/basic-statistics)** | 9-module course covering descriptive statistics, probability, and inferential methods; free to audit | A structured, self-paced course with assignments and quizzes |
| **[edX: Statistics and Data Science (MIT)](https://www.edx.org/learn/statistics)** | University-level courses from MIT, Harvard, and others; free to audit | Advanced learners who want rigorous, institution-backed instruction |

```{tip}
**If you only have time for one resource**, start with **Khan Academy** for foundations or **StatQuest** for targeted help on specific concepts. Both are free, self-paced, and designed for learners who find statistics intimidating.
```

---

### Interactive Tools and Calculators

| Resource | What It Does |
|----------|-------------|
| **[Seeing Theory (Brown University)](https://seeing-theory.brown.edu/)** | Beautiful interactive visualizations of probability, distributions, regression, and Bayesian inference-helps you *see* what the math is doing |
| **[G*Power](https://www.psychologie.hhu.de/arbeitsgruppen/allgemeine-psychologie-und-arbeitspsychologie/gpower)** | Free desktop software (Windows/Mac) for power analysis and sample size calculation; widely cited in social science research |
| **[WolframAlpha](https://www.wolframalpha.com/)** | Type natural-language statistics questions ("standard deviation of 3, 5, 7, 9, 11") and get instant answers with step-by-step solutions |
| **[Social Science Statistics](https://www.socscistatistics.com/)** | Free online calculators for t-tests, chi-square, ANOVA, correlation, effect sizes, and more-just enter your numbers |
| **[JASP](https://jasp-stats.org/)** | Free, open-source desktop statistics software with a point-and-click interface similar to SPSS; supports both frequentist and Bayesian analysis |
| **[jamovi](https://www.jamovi.org/)** | Free, open-source statistics software built on R with a spreadsheet-style interface; designed for researchers who want power without coding |

### Written References and Guides

| Resource | What It Covers |
|----------|---------------|
| **[Statology](https://www.statology.org/)** | Clear, concise written guides on individual statistical concepts and tests-searchable by topic |
| **[Statistics by Jim](https://statisticsbyjim.com/)** | Blog-style explanations of hypothesis testing, regression, ANOVA, and assumptions, written in accessible language |
| **[STHDA: Statistical Tools for High-Throughput Data Analysis](http://www.sthda.com/english/)** | Step-by-step R tutorials for common statistical tests with example code and output |
| **[Laerd Statistics](https://statistics.laerd.com/)** | Detailed guides on choosing and running tests in SPSS; some free content, full access requires subscription |
| **[UCLA OARC Statistical Methods and Data Analytics](https://stats.oarc.ucla.edu/)** | Annotated output guides for SPSS, R, Stata, SAS; "how do I interpret this output?" for nearly every common test |
| **[OpenIntro Statistics (free textbook)](https://www.openintro.org/book/os/)** | Complete, free, downloadable statistics textbook with datasets and exercises; used in many university courses |

| Resource | What It Covers |

```{admonition} A Note on Using These Resources
:class: note
These resources supplement-but do not replace-your course readings and the statistics section of this eBook. When you encounter a concept you don't understand, try looking it up on StatQuest or Khan Academy for a different explanation. Often, hearing the same idea explained two different ways is what makes it click.
```
