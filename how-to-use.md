# How to Use This Book

Welcome! This guide will help you get the most out of your interactive eBook.

---

## Reading the Content

### Navigation

- Use the **left sidebar** to navigate between blocks and chapters
- Use the **right sidebar** (on wider screens) to jump to sections within a page
- Click the **← Previous** and **Next →** buttons at the bottom of each page

### Special Elements

Throughout the book, you'll encounter callout boxes:

```{tip}
Green boxes contain helpful suggestions and practical advice for your methodology chapter.
```

```{important}
Blue boxes highlight key concepts or methodological principles to remember.
```

```{warning}
Yellow boxes alert you to common pitfalls in dissertation methodology writing.
```

```{admonition} Key Reading
:class: seealso
These boxes point you to the specific readings most relevant to the topic at hand.
```

## Interactive Notebooks

Some sections include **Jupyter notebooks** — documents that combine text explanations with runnable Python code. These provide tools like power analysis calculators, assumption-checking dashboards, and coding demonstrations.

### Option 1: Run in Your Browser (Easiest)

Look for the **🚀 Binder** and **Colab** buttons at the top of notebook pages:

- **🚀 Binder**: Launches a free cloud environment (may take a minute to start)
- **📓 Colab**: Opens in Google Colaboratory (requires a Google account)

### Option 2: Download and Run Locally

If you want to work offline:

1. Click the download button (⬇️) at the top of the page
2. Install [Anaconda](https://www.anaconda.com/download) on your computer
3. Open the downloaded `.ipynb` file in Jupyter Notebook or JupyterLab

### Running Code Cells

In a notebook:

1. **Click** on a code cell (the gray boxes with code)
2. **Press** `Shift + Enter` to run it
3. **Wait** for the output to appear below the cell
4. **Experiment** by changing values and re-running!

```{admonition} Don't Panic!
:class: tip
You can't break anything permanently. If something goes wrong, use `Kernel → Restart` to start fresh, or just close the tab and launch again.
```

## Using This Book Alongside the Course

### Before Each Block

1. Read the **block introduction** to understand the themes and deliverable
2. Review the **learning objectives** — what should you be able to do by the end?
3. Work through the **conceptual chapters** alongside your assigned readings

### During Your Work

- Use the **exercises** to draft sections of your block deliverables
- Run the **interactive notebooks** to conduct power analyses, check assumptions, etc.
- Bring questions to your **accountability triad** and to class sessions

### For Your Revision Memos

After each checkpoint weekend, you'll submit a revision memo. The exercises in this book are designed to produce *draft sections* that feed directly into your block deliverables and final methodology chapter.

```{important}
This eBook is a *companion* to your assigned readings, not a substitute. The conceptual chapters synthesize and organize key ideas, but you should engage deeply with the primary sources through Perusall annotations and class discussion.
```
